# HelVox — AI Voice Assistant for Customer Support

## Overview
HelVox is an AI-powered voice assistant designed to provide real-time customer support using natural voice conversations. It helps automate repetitive queries and improves response time.

## Problem
Customer support systems are slow, not scalable, and heavily dependent on human agents. Users often face delays, and teams spend time on repetitive queries.

## Solution
HelVox enables:
- Real-time voice-based interactions
- Automated handling of common queries
- Intelligent responses using a knowledge base (RAG)
- Order tracking and product inquiry support
- Smart escalation to human agents

## How it Works
Voice Input → Speech-to-Text → AI Processing → Response → Text-to-Speech

## Tech Stack
- LiveKit (real-time communication)
- Google Gemini API (AI processing)
- Weaviate (knowledge base)
- Node.js / Python backend
- Next.js frontend
